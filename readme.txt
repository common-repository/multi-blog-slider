Modify image slider to pull posts' featured image from another Wordpress multi-user blog in the same install based on category.

== Description ==

If you have a multi site setup, and want to show posts of other sites in form of slider ?
This plugin  modify content slider to pull posts from another Wordpress multi-user blog in the same install based on category. Go to widget section and put widget named "Multi Blog Slider" where ever you want.

visit https://bhargavb.wordpress.com/2015/07/20/multi-blog-slider/ for detailed info.


== Installation ==

1. Upload the plugin and enable in network.
2. Go to widgets page from “Appearance-> Widgets”. Put the widget wherever you want.

visit https://bhargavb.wordpress.com/2015/07/20/multi-blog-slider/ for detailed info.
